# 👉 프로젝트 소개
**내가 푼 알고리즘 문제 한번에 보기**
* 사용자가 백준이나 프로그래머스에서 푼 알고리즘 문제들을 한번에 확인하고 편리하게 복습 할 수 있는 웹사이트 구현을 목표로 하고 있습니다.
* 클라이언트(프론트) - 서버 - 데이터베이스(비정형 DB) 의 각각의 동작 구조와 패턴, 상호 통신 원리를 공부하기 위한 프로젝트입니다.
